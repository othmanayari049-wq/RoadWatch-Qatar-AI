# RoadWatch Qatar AI

RoadWatch Qatar AI is an open, production-oriented platform for detecting visible road
damage from images and organizing geotagged inspections. The project targets potholes,
longitudinal cracks, transverse cracks, and alligator cracks using the RDD2022 label system.

> The repository is under active development. Detection scores support inspection triage;
> they are not a substitute for an engineering pavement-condition assessment.

## Current status

The domain model and explainable severity heuristic are implemented. API, inference,
dashboard, dataset preparation, evaluation, and deployment layers are added in subsequent
commits.
